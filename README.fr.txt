Cette App permet d'intégrer à Homey les périphérique Royal Gardineer.

Pour l'instant, elle est compatible avec les périphériques suivants  :
    - Irrigation Controller (ZX-7155-675) aussi distribué par PEARL

Si vous avez d'autres périphériques Royal Gardineer que vous désirez intégrer, n'hésitez pas à demander.
